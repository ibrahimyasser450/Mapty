# 🏃 Mapty - Workout Tracker

A workout tracking application that allows users to record their running and cycling activities on an interactive map. Users can add workouts by clicking on the map, view workout details, and store their activities.

## 📌 Overview

Mapty is a JavaScript-based web application built to practice working with:

- Object-Oriented Programming (OOP)
- Geolocation API
- Leaflet Maps
- DOM Manipulation
- Local Storage
- Event Handling
- Modern JavaScript (ES6+)

The application allows users to create workouts by selecting a location on the map and entering workout information.

## ✨ Features

✅ Detect user's current location using Geolocation API  
✅ Interactive map using Leaflet.js  
✅ Add running workouts  
✅ Add cycling workouts  
✅ Automatically calculate workout details:
- Distance
- Duration
- Pace (Running)
- Speed (Cycling)
- Elevation gain

✅ Display workouts on the map  
✅ Click on workouts to move to their location  
✅ Delete workouts  
✅ Save workouts using Local Storage  
✅ Responsive and clean UI design

## 📍 How To Use

1. Allow location access when requested.
2. Click anywhere on the map.
3. Select workout type:
   - Running
   - Cycling
4. Enter workout details:
   - Distance
   - Duration
   - Cadence (Running)
   - Elevation gain (Cycling)
5. Click Enter.
6. Your workout will appear on the map and sidebar.
7. If click on workout will at current location of it.
8. When click delete will delete the workout.
